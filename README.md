# IMAGE EDITING TOOL

## RUN

Run the .ipynb file in supporting ide 



## Installation of Libraries

```bash
pip install tk
pip install numpy
pip install pillow
pip install opencv-python

```
    
## Features

- Resize image
- Rotate image
- Blur image
- Change colorspace
- Edge Detection
- Find Intereset Points
- Connected Components
- Undo
- Revert to original
